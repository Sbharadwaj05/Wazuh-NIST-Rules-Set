# ADR-0006: Apache 2.0 license with named copyright

Date: 2026-05-22
Status: Accepted

## Context
The project will be open source and needs a permissive license with clear ownership.

## Decision
Use Apache License 2.0 and set copyright to Subhash Bharadwaj.

## Consequences
- Users can reuse and distribute with minimal friction.
- Apache 2.0 provides explicit patent grant and protection.
